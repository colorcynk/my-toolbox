## Web Browser
- [Google Chrome](https://www.google.com/intl/ja_jp/chrome/)

  - Mac: `brew install --cask google-chrome`
- [Brave](https://brave.com/ja/)
  - Win: `choco install brave`
  - Mac: `brew install --cask brave-browser`
